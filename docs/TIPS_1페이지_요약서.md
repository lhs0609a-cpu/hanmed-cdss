## 핵심 가치 제안

### 문제
- 한의사 개인 경험에 의존하는 진료 → 품질 편차 발생
- 원로 한의사의 40년 임상 지식이 체계적으로 전수되지 못함
- 신규 한의사의 실전 경험 부족으로 진료 불안감

### 솔루션
**국내 최초 AI 기반 한의학 CDSS**
- 6,000건+ 검증된 치험례 기반 AI 변증/처방 추천
- 자연어 기반 유사 증례 검색
- 양약-한약 상호작용 자동 검사

## 핵심 경쟁력 (Moat)

| 경쟁력 | 설명 |
|--------|------|
| **독보적 데이터** | 40년 경력 원로 한의사의 6,000건+ 치험례 (국내 유일) |
| **기술력** | Claude API + RAG 기반 한의학 특화 AI |
| **선점 효과** | 국내 한의학 CDSS 시장 최초 진입자 |
| **진입장벽** | 양질의 치험례 데이터 확보 어려움 |

## 시장 기회

| 구분 | 규모 |
|------|------|
| **TAM** | 글로벌 CDSS 80억 달러 + 전통의학 3,500억 달러 |
| **SAM** | 국내 한의원 14,000개소, 한의사 25,000명 |
| **SOM** | 수도권 3,500개소 → 초기 500개소 목표 |
| **시장 규모** | 월 10만원 × 500개소 = 연 60억원 |
