## 템플릿 3: AI/딥테크 특화 운영사용

---

**제목**: [TIPS] AI 스타트업 투자 요청 - LLM 기반 도메인 특화 CDSS

---

안녕하세요, [운영사명] 심사역님께

LLM(대규모 언어 모델)을 활용한 도메인 특화 AI 서비스를 개발 중인 양보름입니다.

**[기술적 차별점]**
저희는 Claude API와 RAG(Retrieval-Augmented Generation)를 결합하여 **한의학 도메인에 특화된 임상의사결정지원시스템(CDSS)**을 구축했습니다.

- **벡터 검색**: OpenAI Embedding + Pinecone으로 6,000건 치험례 의미 검색
- **LLM 추론**: Claude API로 증상 기반 변증/처방 추천
- **RAG 파이프라인**: LangChain 기반 정확한 근거 기반 추론

**[왜 한의학인가?]**
- 구조화된 데이터 부재 → AI 적용 시 가치 폭발
- 경쟁자 전무 (양방 CDSS는 영상의학 중심)
- 40년 경력 원로 한의사의 **독점 데이터 확보**

**[기술 스택]**
| 레이어 | 기술 |
|--------|------|
| AI | Claude API, OpenAI Embedding, Pinecone, LangChain |
| Backend | NestJS, FastAPI, PostgreSQL, Redis |
| Frontend | React, TypeScript |
| Infra | AWS, Docker, GitHub Actions |

**[팀]**
- CEO 양보름: 사업총괄
- CTO 이현석: AI/ML 시스템 설계, 풀스택 개발
- 의료자문 이종대 원장: 데이터 제공 및 검증

**[투자 요청]**
2억원+ 투자 및 TIPS R&D 연계 희망합니다.

기술 데모와 아키텍처 상세 설명을 드리고 싶습니다.
